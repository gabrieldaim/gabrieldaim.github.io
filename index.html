Clone Netflix
O "Clone Netflix" é um projeto de desenvolvimento web que consiste na criação de uma plataforma semelhante ao serviço de streaming Netflix. Utilizando as tecnologias ReactJS e CSS puro, o projeto é 100% responsivo e conta com uma conexão à API TheMovieDB para alimentação de conteúdo.

O objetivo do projeto é proporcionar uma experiência semelhante à plataforma original da Netflix, com a possibilidade de exibição de diversos títulos de filmes e séries, bem como informações relevantes sobre cada um deles. Para isso, foi desenvolvido um layout atraente e intuitivo, com fácil navegação e organização de conteúdo.
Blocos de Carnaval
O "Blocos de Carnaval" é um site de pesquisa de blocos de carnaval em sua região. Desenvolvido com HTML e CSS puros, o projeto conta com filtros de pesquisa por localidade e por nome, 100% funcionais graças à utilização de JavaScript.

Com o objetivo de facilitar a busca por blocos de carnaval, o site oferece uma interface responsiva e intuitiva, permitindo que os usuários encontrem facilmente informações sobre os blocos de carnaval mais próximos a eles. O site é ideal para quem deseja se informar sobre a programação de blocos de carnaval em sua região, e também para quem busca aprimorar suas habilidades em desenvolvimento web.
Costs
O "Costs" é um site de gerenciamento financeiro criado com a finalidade de organizar projetos e atividades. Desenvolvido com React JS e styled-components, o projeto oferece uma interface elegante e funcional, permitindo que os usuários gerenciem seus projetos financeiros de forma eficiente e fácil.

Para simular um backend, foi utilizado o JsonServer, permitindo que os usuários possam armazenar e gerenciar dados financeiros diretamente no site. Com o "Costs", é possível criar projetos, adicionar despesas e receitas, definir metas financeiras e monitorar o progresso financeiro ao longo do tempo.
Card Registration
O "Card Registration" é um projeto que permite a inserção de cartões de crédito de forma fácil e segura. Desenvolvido com HTML, CSS e JS puro, o projeto utiliza a ferramenta imask para validar campos, restringir entradas e alterar o layout do site conforme os inputs são preenchidos.

Com o "card-registration", os usuários podem inserir informações de cartões de crédito de forma segura e precisa, sem preocupações com erros de digitação ou informações inválidas. A ferramenta imask permite que o layout do site seja atualizado em tempo real conforme as informações são inseridas, proporcionando uma experiência de usuário mais dinâmica e agradável.
Conversor de moedas
O "Conversor de Moedas" é um projeto de desenvolvimento web que tem como objetivo oferecer uma plataforma de conversão de moedas para os usuários. Utilizando a API economia.awesomeapi.com.br, o projeto permite a conversão de moedas como Real, Dollar, Euro e Libra.

Desenvolvido com HTML e CSS puros, o Conversor de Moedas é 100% responsivo e conta com a ferramenta de gráficos Apex Charts para apresentação dos resultados da conversão em um formato visualmente atraente.
